About

The short version

Fifteen years of hands-on Unreal Engine work — roughly eight of them in Blueprint, four in C++ — covering gameplay, animation, AI, physics and UI systems end to end. The current focus is performance: reading Unreal Insights captures, ranking what's actually costing frame time, and working down the list in order rather than chasing whatever looks scary first.

Before games, precision work of a different kind: repairing jewelry full-time, running laser engraving equipment, managing inventory and dealing directly with customers. It's a strange pairing on paper, but both jobs reward the same thing — noticing the small detail that's actually the problem.

Outside of shipped and client work: more than ten solo-prototyped games since 2015, built purely to keep learning the engine inside out.

Null

An FPS detective-thriller in active development since 2023, built solo in C++ and Blueprint on Unreal Engine 5. Roughly 150 source files across AI, animation, gameplay and architecture — a full-stack testbed, not just a rendering-tech demo.

Systems inside Null
AI & Behavior

Polymorphic enemy hierarchy

Three enemy types share one base class and a common interface contract, so adding a new enemy means extending one class rather than rewriting behavior per-enemy.

Animation

Procedural foot & hand IK

Native C++ AnimInstance drives surface-aware footstep audio across 16 materials, runtime foot locking, and contextual hand IK for in-hand object interaction.

Gameplay Systems

Layered player state machine

Stance, movement speed, fear level and mood run as independent but linked state layers, surfaced live through a HUD indicator that shifts as mental state changes, driving animation and gameplay reactions in real time.

Narrative & Dialogue

Data-driven dialogue with subtitles

DataTable-driven conversation rows carry audio cues, timing and subtitle text, with rows able to trigger gameplay functions directly — chaining dialogue into cinematics and level transitions without hardcoding it in Blueprint.

Interaction & Lighting

Context prompts & dynamic flashlight

Context-sensitive interaction prompts (read, open) and a handheld dynamic light system, built with a hybrid Blueprint-and-C++ workflow — C++ for the underlying logic, Blueprint for rapid iteration on feel.

Architecture

Interface-driven systems

Nine separate interfaces decouple AI, weapons, interaction, HUD and narrative systems. Content is organized into clear functional folders rather than a flat dump, keeping the project navigable as it grows.
